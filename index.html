ParSe v2: predict phase separation potential from the primary sequence

This form allows you to enter a set of sequences and quickly compute a sequence-based prediction of the phase separation potential for each using ParSe v2. The algorithm is described in J. Biol. Chem. 299, 102801 (2023).

Input File

Input a FASTA file containing protein sequences, preferably in the UniProtKB format. The algorithm ignores sequences longer than 10,000 residues or shorter than 25 residues.
